# Doclab - Healthcare Platform

Doclab is a modern, responsive healthcare website template designed to connect patients with medical specialists. Built with clean HTML, CSS, and vanilla JavaScript, it offers a fast and accessible user experience.

![Doclab Preview](./assets/images/hero-banner.png)

## Features

- **Responsive Design**: Fully optimized for mobile, tablet, and desktop devices.
- **Appointment Booking**: Interactive UI for scheduling appointments.
- **Doctor Listings**: Clean grid layout for showcasing medical specialists.
- **Services Showcase**: Detailed view of medical services offered.
- **Blog Section**: Integrated blog layout for health news and updates.
- **Animations**: Smooth scroll reveal and counter animations using vanilla JS.

## Tech Stack

- **HTML5**: Semantic markup for better accessibility and SEO.
- **CSS3**: Custom properties (variables), Flexbox, and Grid for layout.
- **JavaScript**: ES6+ features with no external library dependencies (jQuery-free).

## Setup & Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/imsabbar/doclab.git
   ```
2. Open `index.html` in your browser to view the site.
